## 一、本周趋势总览

今日 GitHub Trending 的核心主题只有一个词：**本地优先（Local-First）**。从 Ollama 到 Open WebUI 到 Osaurus，Top 10 中过半项目围绕"在你自己的硬件上跑 AI"展开。隐私合规驱动的去云化趋势，已从开发者偏好演变为产品需求。

语言分布：Python ~50%、TypeScript ~25%、Rust ~15%、Go ~5%、其他 ~5%。

- **awesome-ai-agents-2026** — 300+ AI Agent 框架目录。当一个领域出现"awesome list"时，说明框架已饱和，机会在垂直场景。

## 三、⚽ AI × 足球统计/分析 专题

### 🏟️ 真实足球分析

#### 1. AgentPitch — AI Agent 足球模拟
- **仓库**：[gangtao/AgentPitch](https://github.com/gangtao/AgentPitch) `[🟢4.2]`
- **语言**：Python/JavaScript/Rust | **Stars**：增长中
- **可用性**：⭐⭐（需 LLM API Key）
- **说明**：球场上的每名球员都是一个 AI Agent，由 LLM 驱动决策逻辑。支持联赛/杯赛/竞技场模式。v0.5.0（2026-07-01），最近一次模拟了 FIFA 2026 半决赛（英格兰 vs 阿根廷）。
- **运行条件**：需要访问 OpenAI/Claude/DeepSeek API。如果是本地跑，需 Ollama + 本地模型。建议至少有 8GB VRAM 或通过 API 调用。

#### 2. AI World Cup — LLM 世界杯预测基准
- **仓库**：[jonaidshianifar/ai-world-cup](https://github.com/jonaidshianifar/ai-world-cup) `[🟡3.2]`
- **语言**：Python/Markdown | **Stars**：新项目
- **可用性**：⭐⭐⭐（标准化提示词，一键运行）
- **说明**：用统一 Prompt 比较 Claude/GPT/Gemini/DeepSeek 等模型对 2026 世界杯的预测。自动评分 + Leaderboard + GitHub Pages 结果站。截止 6 月 12 日有实况排名。
- **运行条件**：只需各模型 API Key，无硬件门槛。

#### 3. World Cup 2026 Prediction — LightGBM 预测管线
- **仓库**：[MouroshK/World_cup_2026_prediction](https://github.com/MouroshK/World_cup_2026_prediction) `[🟡3.8]`
- **语言**：Python | **Stars**：新项目
- **可用性**：⭐⭐（需配置数据管线）
- **说明**：端到端 ML 项目，用 LightGBM + 交叉验证预测淘汰赛晋级。含 Streamlit 交互式 bracket 可视化 + Plotly 图表。最近提交 7 月 16 日。
- **运行条件**：安装 Python + Streamlit + LightGBM 即可本地运行。需从 football-data.org 获取历史数据。

#### 4. Sports CV — YOLOv8 足球 CV 分析
- **仓库**：[abhinav-phi/sports_cv](https://github.com/abhinav-phi/sports_cv) `[🔵2.0]`
- **语言**：Python | **Stars**：持续更新
- **可用性**：⭐⭐（需自己提供比赛视频）
- **说明**：基于 YOLOv8 的足球/板球视觉分析管线，自动输出标注视频、热力图、传球网络和仪表板。
- **运行条件**：需要 GPU（建议 RTX 3060 及以上）+ 输入视频。纯 CPU 可以跑但非常慢。

#### 5. AI Football Tactical Analysis — 深度学习战术分析
- **仓库**：[DevaNandanJS/Football-Tactical-Analysis-AI-Coaching](https://github.com/DevaNandanJS/Football-Tactical-Analysis-AI-Coaching) `[🔴1.5]`
- **语言**：Python | **Stars**：新项目
- **可用性**：⭐⭐（需要视频输入 + GPU）
- **说明**：YOLO + BoTSORT 追踪 + Homography 映射，从视频提取球员统计、热力图、传球网络并生成比赛报告。专业级管线。
- **运行条件**：GPU 推荐。Homography 标定需要一定计算机视觉知识。

#### 6. Agentic FC — AI Agent 足球管理模拟
- **仓库**：[gaemi/agentic-fc](https://github.com/gaemi/agentic-fc) `[🟢4.0]`
- **语言**：Go | **Stars**：新项目
- **可用性**：⭐⭐（Go 环境 + MCP 配置）
- **说明**：开源足球管理模拟，AI Agent 通过 MCP 协议作为自主经理，做出转会、战术、阵容决策。TUI 控制台观看比赛。v0.1.0（7 月 9 日）。
- **运行条件**：需 Go 编译环境 + LLM API 接入。种子确定世界，适合做比赛推演实验。

#### 7. Three-Agent 世界杯预测系统

## 四、今日 Top 3 推荐

1. **🥇 Ollama + Open WebUI 组合** — 本地 AI 基础设施标准配置。Ollama 负责模型运行，Open WebUI 提供前端。内网可搭建的私有 AI 助手，适合团队使用。
